== Description ==

Mighty Media Cleaner finds the image files which have not been used within a post or page and displays them for deletion from both your uploads directory and database.

Don't forget to have a proper backup of your files and database before using this plugin. You can use Backup and restore features that available in the plugin.

Two way to scan media: Auto scan (full scan) and find the usage of one specific file.

== Installation ==

1. Upload `mighty-media-cleaner` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Dashboard -> Mighty Media Cleaner and check the appropriate options
